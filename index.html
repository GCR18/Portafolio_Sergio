Proyecto Destacado

Sistema de Gestión de Horas de Trabajo Formativo

El proyecto digitaliza procesos que antes se realizaban de forma manual, eliminando el uso de papel, reduciendo errores humanos y mejorando la rapidez en la entrega de resultados médicos.

El proyecto consiste en el desarrollo de un sistema web de gestión y consolidación automatizada de horas de trabajo formativo para la Universidad Adventista de Bolivia. Su objetivo es reemplazar los registros manuales y dispersos actualmente utilizados, por una plataforma centralizada que permita registrar y administrar las horas laborales realizadas por estudiantes y personal académico.
